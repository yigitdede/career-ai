# Arşiv

Bu klasördeki dosyalar **aktif geliştirme dışı** referans kodlarıdır.

| Klasör / dosya | Açıklama |
|----------------|----------|
| `streamlit/` | İlk MVP arayüz (Streamlit). Yerine `../frontend/` Laravel kullanılıyor. |
| `laravel/` | Laravel monolith denemesi (API + UI tek projede). Yerine ayrık mimari: `../backend/` + `../frontend/`. |
| `backend/`, `frontend/`, `web/` | Eski dizin kopyaları; güncel kod repo kökündeki `backend/` ve `frontend/` altında. |

**Güncel stack:** FastAPI (`backend/`) + Laravel (`frontend/`)
