## Yeni sürüm oluşturma

1. Şablonu kopyala: [_sablon-yeni-surum.md](_sablon-yeni-surum.md)
2. Dosya adı: `YYYY-MM-DD-vNNN-kisa-baslik.md` (ör. `2026-07-06-v002-ilan-eslestirme.md`)
3. `vNNN` bir önceki sürümden +1
4. Bu README tablosuna yeni satır ekle
5. Yeni dosyada **«Önceki sürüme göre değişiklik»** bölümünü doldur
